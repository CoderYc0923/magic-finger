# my-app

Magic Finger~~~

## Project Design Scheme

```bash
1. 固定功能： 鼠标左右单击、双击；鼠标拖动；鼠标滚轮
2. 简约的界面：首页放置开始按钮和说明按钮即可
2.1 开始按钮说明： 点击开始，即运行功能
2.2 说明按钮说明： 点击说明，弹出弹窗，展示各手势介绍
```

## Project Setup

### Install

```bash
$ yarn
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
